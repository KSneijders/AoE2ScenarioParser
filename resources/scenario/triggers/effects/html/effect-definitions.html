Effect Definitions (101 effects)

0 none This effect is empty. It doesn't even have 'splash' or evolve...
no attributes
1 change_diplomacy This effect can be used to change the diplomacy stance of the source players with the target player. It does NOT change the stance both ways.
diplomacy_state DiplomacyState The diplomacy stance to set.
source_player PlayerId The player whose stance will be changed.
target_player PlayerId The target player whose diplomacy stance will be changed for the source player.
2 research_technology This effect can be used to automatically research a technology for the specified player.
source_player PlayerId The player for whom the technology will be researched.
technology_id TechInfo The technology which will be researched for the specified player.
force_technology bool When enabled, force the selected technology to be researched, regardless if the civilization has access to it or not.
3 send_chat This effect can be used to send a chat message to the specified player.
source_player PlayerId The player who will be receiving the message.
str_id int The string ID to use instead of the message.
message str The message to send to the player.
sound_name str The name of the sound to play when the message is sent.
4 play_sound This effect can be used to play a sound at a specific location or globally for the specified player.
source_player PlayerId The player who will hear the sound effect.
location Tile The tile to play the sound at. When not set, the sound will be played globally.
location_object_ref Unit The unit whose position and rotation will be used to play the sound at. When not set, the sound will be played globally.
global_sound bool When enabled, the sound will be played regardless if the target is on screen. This is useful for existing game sounds that are considered '3D'. '3D' sounds are game sounds based on coordinates. '2D' sounds are sounds like GUI, button click or notification sounds. default: False
sound_name str The name of the sound to play.
5 tribute This effect can be used to send resources from one player to another.
quantity int The amount of the specified resource to send.
resource Attribute The resource that will be sent.
source_player PlayerId The player who will be sending the resources.
target_player PlayerId The player who will receive the resources.
6 unlock_gate This effect can be used to unlock specific gates.
selected_objects list[Unit] The gates to be unlocked.
7 lock_gate This effect can be used to lock specific gates.
selected_objects list[Unit] The gates to be locked.
8 activate_trigger This effect can be used to activate a specific trigger.
trigger Trigger The trigger to be activated.
9 deactivate_trigger This effect can be used to deactivate a specific trigger.
trigger Trigger The trigger to be deactivated.
10 ai_script_goal This effect can be used to communicate with an AI player by setting an AI Trigger number. The AI Trigger number can be detected inside an AI script using: `event-detected trigger NUMBER`
ai_script_goal int The AI Trigger number to set
11 create_object This effect can be used to create a unit (including buildings, heroes etc.) for the specified player
object_unit_id UnitInfo|BuildingInfo|HeroInfo|OtherInfo|int The unit type to be created.
source_player PlayerId The player whose unit will be created
location Tile The tile where the unit will be created
item_id int The type of the unit that is selected in the dropdown
facet int The rotation of the created unit. This can be any integer between 0 and 15, with 0 looking towards the right of the screen and 8 looking towards the left of the screen. With increasing values moving counter clockwise.
disable_sound bool When enabled, disable the creation sound of the object default: False
12 task_object This effect can be used to order units to perform a specific action at a target location or on a target object.
object_unit_id UnitInfo The type of unit to task
source_player PlayerId The player whose units will be tasked
location Tile The tile to send the tasked units to
location_object_ref Unit The target unit (as if it was right clicked)
area Area The area in which units will be tasked. When not set, units across the entire map are tasked
object_group ObjectClass The units with this class will be affected by this effect
object_type ObjectType The units of this type will be affected by this effect
action_type ActionType The type of action to perform on the affected units
selected_objects list[Unit] The units to be affected by this effect. When defined, overwrites all other unit filters, like area selection, unit type, object type etc.
disable_garrison_unload_sound bool When enabled, disables the sound that plays when unloading garrisoned units
max_units_affected int The maximum number of units affected by this effect
issue_group_command bool When enabled, will issue a task as a group formation task, instead of every unit individually
queue_action bool When enabled, the set task will be shift-queued
